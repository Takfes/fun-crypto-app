### How to run strategy
> python backtesting.py futures15 ETHUSDT dic 10000 0.5

### Strategy Options
* futures15 -> table name
* ETHUSDT -> pair
* dic -> strategy to test
> other options : 'ma','macd','bnh','stoc','triple','dip','bearx','dic','3h'
* 10000 -> total budget
* 0.5 -> risk percentage (applied on total budget)
